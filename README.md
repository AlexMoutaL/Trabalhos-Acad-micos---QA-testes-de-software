# Trabalhos-Acad-micos---QA-testes-de-software
Aqui se encontram algumas atividdades acadêmicas feitas durante o curso de DESENVOLVIMENTO EM TESTES DE SOFTWARE
Utilizando o Selenium IDE, foram feitos alguns testes básicos de interface e funcionalidades de um site e-commerce (Mercado livre).
Criando 4 testes baseados cenários ao todo, e utilizando o Assert e Verify em 2 dos cenários criados.
Além disso no arquivo de slide é possível ver a descrição dos cenários criados com mais detalhes.
Ainda no arquivo de slide, é possível ver o uso do Jira para genrrenciamento dos testes, tela do backlog, uso de kanban e scrum.
Todos os testes passaram com sucesso, nenhum bug foi encontrado!!
